How to post Interactive Kalman Filter on Website:
1. On Godaddy, Access cPanel in my Linux Hosting https://www.godaddy.com/help/access-cpanel-in-my-linux-hosting-40972
    1.1 Go to Godaddy Product page
    1.2 Select Web Hosting, and then select Manage for the Linux Hosting you want to use.
    1.3 In the top-right, select cPanel Admin
2. Once in the Cpanel File Manager, upload a zip file of the interactive-kalman-filter repo to the public_html directory (can't upload just a folder), then extract this into a folder named interactive-kalman-filter. 
3. Now you can access it via https://arthurlovekin.com/interactive-kalman-filter.
